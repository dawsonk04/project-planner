Project Tracker - Getting Started

Welcome to Project Tracker! This application helps you manage your projects and tasks from start to finish.

Quick Start

Windows Users

  1. Double-click the install.bat file
  2. The script will check if Node.js is installed, install dependencies, and start the application
  3. The application will open in your default browser at http://localhost:3000

macOS/Linux Users

  1. Open Terminal in the project directory
  2. Make the script executable: chmod +x install.sh
  3. Run the script: ./install.sh
  4. The application will open in your default browser at http://localhost:3000

Prerequisites

Before you start, make sure you have the following installed:

  • Node.js 18+ - Download from nodejs.org
  • npm (comes with Node.js) or yarn

Manual Installation

If the quick start scripts don't work for you, follow these steps:

  1. Open a terminal/command prompt
  2. Navigate to the project directory:
    cd path/to/project-planner
  3. Install dependencies:
    npm install
    or
    yarn install
  4. Run the development server:
    npm run dev
    or
    yarn dev
  5. Open http://localhost:3000 in your browser

Using the Application

Creating a Project

  1. Click "Add New Project" button
  2. Fill in the project details
  3. Click "Create Project"

Managing Tasks

  1. Click "View Tasks" on a project card
  2. Add, edit, or delete tasks for that project

Updating Project Status

  1. Click "Edit" on a project card
  2. Update the status and other details
  3. Click "Update Project"

For more information, please refer to the README.md file.
